## ✨ What is Nebula Nb?

**Nebula-NB** is a modern version control system (VCS) CLI developed by **Udit N. Khare & Swara Verma**.

---

## 🚀 Features

* ⚡ Update-based version control
* 🔄 Snapshot & advanced revert system
* 👥 Team collaboration
* ☁️ Server-based sync
* 🧩 Beginner-friendly workflow
* 📦 Scales to large projects

---

## 📦 Installation

```bash
pip install nebula-nb
```

---

## 🚀 Quick Start

```bash
nb startproject [project name]
cd [project name]
nb startupdate [update name]
work on project inside workspace folder
nb publish [update name]
```
